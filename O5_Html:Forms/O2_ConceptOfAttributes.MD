## Concept of Attributes 

~~~
Attributes or many people say properties of an html element 
provides extra functionalities and information to the html elements 
~~~

~~~
Example 

<input type="text" placeholder="Your Name">

and 

<input type="submit" value="Click me to submit" disabled>

here the value represents the text value which will be displayed in the input box 
and also it has a value 


whereas in case of disabled attribute there's no value concerned with it 
~~~
