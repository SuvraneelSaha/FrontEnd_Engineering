## Html level validation attributes 

~~~
html level validation attributes are the atts 
which are already specified and its usecases is known 
~~~

~~~
Example 
<input type="text" placeholder="your name" minlength="10" maxlength="100" name="firstName" autofocus>

    <!-- autofocus when we first open the webpage this input will be highlighted  -->

    <input type="text" placeholder="address" value="ABC Nagar" name="address" size="55">

    <!-- size is same as the width of an input in css its plain width  -->

    <input type="number" name="age" min="10" max="100">

    <!-- value must be equal to or greater than 10 will be prompted when we try to submit the form 
    where the age is given below 10 and same case for the max -->

    

    <input type="submit" name="submit">

~~~
