## Placeholder , Value and Disabled attributes


~~~
Placeholder : 
it acts as a temporary text inside of a text box or an input box 
~~~

~~~
Value Attribute : 
value attribute is the actual text that is present inside of an input text box or any text box 
~~~

~~~
disabled attribute : 
specifically used with the submit buttons so that whenever an user tries to submit a form 

and if all the conditions are not satisfied then the button will be blurry and if all the conditions are 
satisfied then button will be enabled  
~~~
