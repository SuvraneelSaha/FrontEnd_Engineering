## Selectors , universal and tag selectors 

~~~
What is Selectors in css ? 

it is a pattern to select an element or an html element 
for styling in the css
~~~

~~~
What is a universal selector ? 

Universal Selector or * selector selects all the elements classes or ids 
manually (automatically we dont need to do anything else)

then we can override the * selector by defining the individual styles for elements or classes 
~~~

~~~
What is a tag selector or an element selector ? 

Element Selector is used to select a specific element or html element 

and style them 
~~~

~~~
What Properties can be defined inside of the * selector ? 

basically margin and padding can be  changed to make the document a border box 

otherwise its frugal to make changes like background color or font size or color etc 


~~~
