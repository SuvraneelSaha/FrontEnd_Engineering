## Font Weight and Gradients in CSS

### Font Weight
~~~
font-weight:bold;
font-weight:lighter;
font-weight:500;

~~~

### Gradient 
~~~
background-image:linear-gradient(to bottom,#ff0000,#00ff00);

background-image:linear-gradient(to right,#ff0000,#00ff00);

background-image:radial-gradient(circle,#ff0000,#00ff00);


~~~
