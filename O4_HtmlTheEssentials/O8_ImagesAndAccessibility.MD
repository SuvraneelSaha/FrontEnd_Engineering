## Images and Accessibility 

~~~
Images in html can be used via the 
img element 

the src attribute is used to specify the source location of the image 

and the alt attribute is used to specify the alternative text for screen readers .
Screen readers are used by the persons with disability example blindness , etc . 
So for them we made it . 
~~~

~~~
What are attributes : 
attributes of the html elements gives extra information or meta data about the html element 
example 
<img src ="./local/usr/Downloads/cat.png" alt="cute cat">

these src and alt are attributes which specify extra information about the html element img 
~~~
