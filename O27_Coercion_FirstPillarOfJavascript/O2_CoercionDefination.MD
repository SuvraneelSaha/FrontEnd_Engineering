## Coercion Definition  

~~~

Whenever we do an operation , based on the input we can actually convert the input for operations . 

ie we can convert the the type of the input 

The conversion can be done manually by us 
which is known as explicit typecasting or 
explicit type conversion 

~~~~

~~~~

the conversion that is done automatically is known implicit type conversion , this is also known as Coercion 

coercion is also known as implicit type casting or automatic type conversion . 

~~~
